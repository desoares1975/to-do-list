# Aplicação 2 - To-do list

## Getting started:

```bash
$ git clone git@github.com:desoares1975/to-do-list.git
```
### After clonning on the same folder:

```bash
$ cd to-do-list
```
```bash
$ npm install
```

Save the file in the main folder .env.sample as .env
```bash
$cp .env.sample .env
```

## Starting application

```bash
$ node index.js
```
